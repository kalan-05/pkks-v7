# Требования и вопросы по хостингу

Статус: требуется подтверждение заказчика перед установкой на боевой хостинг.

## Домен

- Какой финальный домен будет использоваться для сайта?
- Кто управляет DNS-записями домена?
- Кто будет подтверждать переключение домена на новый сайт?
- Нужно ли сохранить старые DNS-записи или поддомены?

## Хостинг

- Название хостинг-провайдера.
- Панель управления хостингом.
- Версия PHP.
- Тип веб-сервера: Apache, Nginx или связка Apache/Nginx.
- Поддерживается ли `.htaccess`.
- Поддерживается ли rewrite на уровне выбранного тарифа.
- Какая корневая папка должна отдавать сайт.
- Какой способ загрузки файлов разрешён: панель, FTP, SFTP, SSH или другой безопасный канал.

## Права и runtime

- Какие папки доступны на запись PHP-процессу.
- Можно ли безопасно хранить runtime-файлы вне публичной корневой папки.
- Где должны храниться логи, временные файлы и резервные копии.
- Нужны ли отдельные права для папок загрузки изображений сотрудников.

## SSL и почта

- Кто выпускает и продлевает SSL-сертификат.
- Включён ли редирект с HTTP на HTTPS.
- Нужна ли отправка почты с сайта.
- Если почта нужна, какие SMTP-настройки разрешено использовать и кто их предоставляет.

## Доступы

- Если установка выполняется нашей стороной, нужны временные доступы только на период работ.
- Пароли, токены и ключи нельзя передавать в обычной переписке.
- Доступы должны передаваться через согласованный защищённый канал и отзываться после завершения установки.
